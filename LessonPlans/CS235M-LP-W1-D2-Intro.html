Development Environment

We will be using Visual Studio with the Xamarin mobile development extensions.
  • Set up Visual Studio and Tools
    • Include "Mobile Development" in Visual Studio setup
    • The Google Android SDK will be installed with the above selection
    • If the Android Virtual Device Manager provided in the Android SDK isn't working, you can download the Xamarin Device Manager
    • Optional: Install the Microsoft Android Emulator if using Windows 10 Pro or above
  • Tour the Android SDK
  • Set up an emulator

C# Review

  • Attributes
    • Associate metadata, or declarative information, with code
    • Your program can examine its own metadata
  • Delegates
    • Facilitates anonymous methods
    • A late binding mechanism - the algorithm (code) is provided at run-time.
    • Used in Activities to define event handlers
