Preparation for Distribution

These steps need to be completed regardless of the distribution method. Even to dis

  1. Ensure the app has an icon - set in the project properties, "Android Manifest"
  2. Set the application version - same as above
  3. Specify the best linking option - in project properties, "Android Options", Linking, choose "Sdk Assemblies only"
  4. Enable pro-guard - prevents reverse engineering and shrinks the APK. Same property page as above
  5. Archive for publishing - Go to "Build", "Archive..." - The archive (APK) will be built
  6. Sign the APK - Go to "Tools",  "Archive Manager" (you are already there if you just did step 5)
    • Click the Distribute button
    • Choose a type of distribution (Ad-Hoc or Google Play)
    • Click + to create a new signing identity (You must fill in all the fields)
    • Save the APK to a folder of your choice
  7. The APK file is now ready for distribution

Reference: Xamarin Guide: Preparing an Application for Release

Ad-Hoc Distribution

Prepare the APK for distribution using the steps above.
Next, do one of the following:
  • Copy the APK file to a device using a USB cable
  • Put the APK on a flash drive
  • Email a copy of the APK
  • Post the APK on a website for download
    • diawi is a website developers can use to distribute their APKs for testing.
    • Use your own web site - just upload the APK to the server and add a link to a web page.

Users will need to enable installation of apps from outside an app store

  • Pre-Oreo (API leve 25 or lower): open the settings on their device and go to "Security", "Unknown Sources" to enable installation of the app.
  • Oreo (API level 26 or above): When you tap the APK file to open it on your device, you will be presented with a pop-up dialog asking you to approive instalation of an app from an unknown source.

Reference: Xamarin Guide: Publishing Independently

Distributing Apps via the Google Play Store

Launcher icons are required in order to publish your app to Google Play.
  • Launcher icons required for all screen resolutions
    • hdpi
    • mdpi
    • xhdpi
    • xxhdpi
    • xxxhdpi
  • One easy way to create icons: Android Asset Studio Web Site
(Note to myself: to log in to the Google Play Console use my personal Google account)

Alpha and Beta Testing via the Play Store

On the Google Play Console:

  • Select an app that has been uploaded
    • Select "App releases"
      • Click on "Manage alpha" or "Manage beta"
        • Choose a testing option:
          • Open
          • Closed
          • Via Google Groups or Google+
