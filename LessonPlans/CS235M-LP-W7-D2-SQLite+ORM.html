Installing SQLite in Windows

  1. Download:
    •     sqlite3.def - might be needed if you are compiling and linking source code that uses the SQLite DLL
    •     sqlite3.dll - the actual databse engine
    •     sqlite3.exe - a CLI for SQL commands
  2. Put sqlite3.dll in one of these folders:
    • a new folder of your choice. Add the folder to the Windows Path variable
    • a system folder. The choice depends on the app or apps that will be using SQLite. If you are using it with a console app built in Visual Studio the default is 32 bit, but you can change that in the project build settings.
      • Windows\SysWOW64 for the x86 (32 bit) DLL - this sounds backwards, but the 64 bit version really goes here
      • Windows\System32 for the 64 bit DLL - yes the 32 bit version really goes here
    • the same folder with the application that will be using SQLIte
Reference: Installing and Using SQLite on Windows - John Atten
